# Pokémon Illustration Rare Finder

Eine persönliche Referenzseite für alle Pokémon Illustration Rare Karten.

## Features
- 🔍 Suche nach deutschen und englischen Pokémon-Namen
- 🗂 Alle 1025 Pokémon (Gen I–IX) nach Pokédex-Nummer sortiert
- 📦 Anzeige in welchem Booster Pack eine Karte zu finden ist
- 📱 Responsive Design für Mobile, Tablet und Desktop
- 🃏 Platzhalter für Pokémon ohne Illustration Rare

## Daten
- **680 Illustration Rare Karten** aus 20 verschiedenen Sets
- **501 Pokémon** haben mindestens eine IR-Karte
- **89 Trainer-Karten** als eigene Kategorie

## Verwendete APIs
- [Pokémon TCG API](https://pokemontcg.io/) – Kartendaten und Bilder
- [PokéAPI](https://pokeapi.co/) – Pokémon-Namen (DE/EN)
