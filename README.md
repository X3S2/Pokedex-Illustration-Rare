# Pokémon Kartenfinder

Eine persönliche Referenzseite für Pokémon Sammelkartenspiel-Karten mit künstlerischen Seltenheitsstufen.

## Features
- Suche nach deutschen und englischen Pokémon-Namen
- Alle 1025 Pokémon (Gen I–IX) nach Pokédex-Nummer sortiert
- Filter nach Generation und Seltenheitsstufe
- Anzeige in welchem Booster Pack eine Karte zu finden ist
- Responsive Design für Mobile, Tablet und Desktop
- Klick auf eine Karte öffnet eine Detailansicht

## Daten
- **719 von 1025 Pokémon** haben mindestens eine Karte
- **832 Trainer-Karten** als eigene Kategorie
- 306 Pokémon haben keine künstlerische Karte (nur Common/Uncommon im TCG)
